# fajarjulyana.github.io
Repository fajarjulyana.github.io berisi source code situs pribadi Fajar Julyana yang menampilkan portofolio, proyek, dan dokumentasi pengembangan software. Situs ini menjadi pusat informasi karya Fajar di bidang C++, audio DSP, dan plugin VST, dengan desain modern, responsif, serta mudah diakses.
